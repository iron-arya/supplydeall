# Supply Mission 🚁📦

## Project Overview
Supply Mission is a simple physics-based simulation built using **p5.js**, **p5.play**, and **Matter.js**.  
The project demonstrates how physics engines can be integrated with JavaScript graphics libraries to simulate real-world behaviour like gravity and object dropping.

In this project, a helicopter carries a supply package and drops it when the user presses the **DOWN ARROW key**. The package then falls realistically using Matter.js physics.

---

## Features
- Helicopter carrying a supply package
- Physics-based package dropping using Matter.js
- Gravity simulation
- Keyboard interaction
- Sprite rendering using p5.play
- Simple game-style environment

## How it Works
1. The helicopter and package are displayed using sprites.
2. The package is connected to a static Matter.js body.
3. When the **DOWN ARROW** is pressed:
   - The package body becomes dynamic.
   - Gravity pulls the package down.
4. The package falls and lands on the ground.

---

| Key | Action |
|-----|--------|
| ↓ Down Arrow | Drop the package |
